How To Use

Work through the content found in the left navigation bar, from top to bottom. Make sure to look at your checklist so you know where you need to get to for each day. Topics and concepts are divided into chunks that require you to tinker, learn, apply, and create:

  1. Spot the Pattern: These are pages where you tinker with concepts. Think of them as mini-puzzles where you'll have to see if you can find patterns to help you get an introduction to the concepts. Don't skip these pages, as they are what set the ground work for new neural connections.
  2. Reading Content: These are often identifiable by the fact that their title is just the concept or topic you will learn. These are the readings and videos that explain concepts in more detail. Think of these as where you read to learn more and where you reference back to really learn.
  3. Algo App: These are assignments that direct you to completing portions of the Algorithm App that apply what you learned.
  4. Code Challenge: These are assignments where you will have to create something - usually in Visual Studios Code - and submit via zip file.
Work through the next few pages to get a feeling for what each type of content is like.
